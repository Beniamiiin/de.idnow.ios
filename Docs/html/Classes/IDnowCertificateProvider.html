IDnowCertificateProvider Class Reference

Inherits from NSObject
Declared in IDnowCertificateProvider.h

Overview

classes that implement this interface are used to provide custom DTLS certificates used by the WebRTC connection.

In order to use custom certificates please subclass IDnowCertificateProvider and add an instance of your new class to the IDnowSettings:

IDnowSettings.certificateProvider = >>YOUR_SUBCLASS<<
