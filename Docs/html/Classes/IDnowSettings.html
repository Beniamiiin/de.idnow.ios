IDnowSettings Class Reference

Inherits from NSObject
Declared in IDnowSettings.h

Overview

This class is used to instantiate an instance of IDnowController. It lets you customize the identification process. At least a transactionToken and a companyID have to be set before calling [IDnowController initialize].
